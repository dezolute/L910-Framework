# L910-Framework

## Описание файлов

Проект содержит 6 JSON файлов, организованных по трем тематическим разделам:

Музыкальная индустрия

`artists.json` - Музыкальные исполнители
Содержит информацию о музыкальных группах и исполнителях.

**Поля:**
- `id` - уникальный идентификатор
- `name` - название группы/исполнителя
- `country` - страна происхождения
- `genre` - музыкальный жанр
- `foundedYear` - год основания
- `isActive` - активна ли группа сейчас
- `memberCount` - количество участников
- `albums` - список альбомов
- `biography` - краткая биография

`concerts.json` - Концерты и мероприятия
Содержит информацию о предстоящих концертах.

**Поля:**
- `id` - уникальный идентификатор
- `title` - название мероприятия
- `artistId` - ссылка на исполнителя (связь с `artists.json`)
- `venue` - место проведения
- `date` - дата и время проведения
- `price` - цена билета
- `availableTickets` - количество доступных билетов
- `isSoldOut` - распроданы ли все билеты
- `genres` - музыкальные жанры мероприятия
- `description` - описание мероприятия

**Связь:** `concerts.artistId` → `artists.id`

### 2. **Косметическая индустрия**

#### `brands.json` - Бренды косметики
Содержит информацию о косметических брендах.

**Поля:**
- `id` - уникальный идентификатор
- `name` - название бренда
- `country` - страна производства
- `foundedYear` - год основания бренда
- `isOrganic` - органическая ли косметика
- `hasCrueltyFree` - тестируется ли на животных
- `certifications` - сертификаты качества
- `productCount` - количество продуктов в линейке
- `website` - официальный сайт
- `rating` - рейтинг бренда
- `description` - описание бренда

`products.json` - Косметические продукты
Содержит информацию о конкретных косметических продуктах.

**Поля:**
- `id` - уникальный идентификатор
- `name` - название продукта
- `brandId` - ссылка на бренд (связь с `brands.json`)
- `category` - категория продукта
- `price` - цена в рублях
- `stock` - количество на складе
- `isAvailable` - доступен ли для покупки
- `expiryDate` - срок годности
- `ingredients` - список ингредиентов
- `volume` - объем/вес в мл/г
- `rating` - рейтинг продукта
- `description` - описание продукта

**Связь:** `products.brandId` → `brands.id`

### 3. **Туристическая индустрия**

`hotels.json` - Отели
Содержит информацию об отелях.

**Поля:**
- `id` - уникальный идентификатор
- `name` - название отеля
- `country` - страна расположения
- `city` - город
- `address` - адрес
- `stars` - количество звезд
- `hasPool` - наличие бассейна
- `hasWifi` - наличие Wi-Fi
- `roomCount` - количество номеров
- `amenities` - удобства и услуги
- `pricePerNight` - цена за ночь
- `rating` - рейтинг отеля
- `description` - описание отеля

`tours.json` - Туристические туры
Содержит информацию о туристических турах.

**Поля:**
- `id` - уникальный идентификатор
- `title` - название тура
- `destination` - место назначения
- `hotelId` - ссылка на отель (связь с `hotels.json`)
- `departureDate` - дата выезда
- `returnDate` - дата возвращения
- `price` - цена тура
- `availableSeats` - доступные места
- `isAvailable` - доступен ли тур
- `includedServices` - включенные услуги
- `rating` - рейтинг тура
- `duration` - продолжительность в днях
- `description` - описание тура

**Связь:** `tours.hotelId` → `hotels.id`
